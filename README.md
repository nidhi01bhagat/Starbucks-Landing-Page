## My Approach (Built from Scratch)
This project was built during my early learning phase, where I implemented everything step by step to understand core frontend concepts in depth.
Key areas I focused on:
 1.Understanding layout systems using Flexbox and CSS Grid
 2.Writing semantic HTML for better structure and readability
 3.Implementing responsiveness using media queries
 4. Debugging layout and alignment issues through trial and iteration

## Challenges Faced
* Making the layout responsive across different screen sizes
* Maintaining consistent alignment and spacing
* Structuring CSS in a clean and manageable way

## What I Learned
* How real-world landing pages are structured
* Importance of writing clean and maintainable CSS
* Fundamentals of responsive design and UI thinking
